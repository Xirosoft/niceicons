NPM

Note: If you'd like to use Xiroicon with a CDN, you can skip this NPM (installation) step.

npm install xiroicon --save
import 'xiroicon/xiroicon.css'

import CSS to your main.js



CDN

Copy the following code and add it to the <head> tag of your html document.

<link rel="styleheet" href="https://xirosoft.github.io/xiroicon/icon/xiroicon.min.css" />


Download Manually

Download xiroicon.css file, add it to the <head> tag of your html document.



Use

Add icon with class name, class name rule: xiroicon xi-{name}

<i class="xiroicon xi-add"></i>
<i class="xiroicon xi-arrow-left">
